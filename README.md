# InformedVoter

A nonpartisan civic information platform for US voters — the "Wikipedia for government." Research your representatives, track legislation, follow Supreme Court cases, explore campaign finance data, and exercise your rights — all in one place, without partisan spin.

**Live:** [informed-voter.vercel.app](https://informed-voter.vercel.app)

## Features

### Interactive State Map
- Click any state on the SVG map to explore its civic information
- Dropdown search for quick state selection
- Remembers your selection across visits

### Representatives & Elected Officials
- **US Senators** — Profiles, policy positions, bill sponsorships, recent votes
- **US Representatives** — District-level representation with full voting records
- **Governors** — State executive profiles and policy positions
- **Candidate Profiles** — Detailed pages with biography, policies, voting history, and campaign finance

### Legislation Tracking
- **Federal & State Bills** — Browse, filter by chamber, status, and keyword search
- **AI-Powered Summaries** — Plain-English bill summaries generated by Claude AI
- **Hidden Clause & Rider Detection** — AI flags unrelated provisions buried in legislation
- **Bill Detail Pages** — Full text, sponsor info, vote history, and AI analysis

### Campaign Finance Transparency
- **Top Donors** — Who's funding each candidate, with employer and occupation data
- **Industry Breakdown** — Contributions grouped by industry sector
- **Spending Analysis** — Campaign expenditures by category (media, payroll, travel, consulting)
- **Contribution Size Distribution** — Small-dollar vs. large-dollar donor breakdown
- **Independent Expenditures** — Outside spending for or against candidates
- Data sourced from the Federal Election Commission via OpenFEC

### Supreme Court (Judicial Branch)
- **SCOTUS Dashboard** — Active justices, pending cases, and recent decisions
- **Justice Profiles** — Voting records, ideology scores, appointment history
- **Financial Disclosures** — Gifts, paid travel/trips, investments, and annual disclosure PDFs
- **Case Detail Pages** — Docket info, oral argument dates, vote breakdowns (majority vs. dissent)
- **AI Case Analysis** — Plain-English case summaries and public impact analysis
- Data sourced from Oyez API and CourtListener (Free Law Project)

### Federal Agencies
- **Agency Directory** — Browse federal agencies by category (defense, health, economy, environment, etc.)
- **Agency Profiles** — Mission, leadership, official website links

### Voter Action Center
- **Registration Info** — State-specific registration requirements and deadlines
- **Voter ID Requirements** — What you need to bring to the polls
- **Early Voting & Absentee Ballots** — Dates, rules, and request links per state
- **Voter Rights** — Know your rights at the polling place
- **Media Literacy Tips** — How to evaluate political news and identify bias

### Polling Place Finder
- Address-based lookup for your nearest polling location
- Powered by Google Civic Information API

### Candidate Comparison
- Side-by-side policy comparison for candidates running for the same office

## Data Sources

| Source | Data Provided |
|--------|--------------|
| [Congress.gov API](https://api.congress.gov/) | Bills, members, voting records |
| [LegiScan API](https://legiscan.com/legiscan) | State + federal bills, full text, roll calls |
| [OpenFEC API](https://api.open.fec.gov/) | Campaign finance, donors, expenditures |
| [Google Civic Information API](https://developers.google.com/civic-information) | Elections, polling places, ballot info |
| [Oyez API](https://www.oyez.org/) | SCOTUS cases, justices, oral arguments |
| [CourtListener API](https://www.courtlistener.com/) | Justice financial disclosures (Free Law Project) |
| [BigDataCloud](https://www.bigdatacloud.net/) | Reverse geocoding |

## Automated Data Sync

Cron jobs keep data fresh automatically:

| Job | Schedule | Source |
|-----|----------|--------|
| Sync federal legislators | Daily 6:00 AM | Congress.gov |
| Sync SCOTUS cases & justices | Daily 5:00 AM | Oyez + CourtListener |
| Sync bills | Daily 7:00 AM | Congress.gov + LegiScan |
| Sync voting records | Daily 8:00 AM | Congress.gov |
| AI bill analysis | Daily 12:00 PM | Anthropic Claude |
| AI case analysis | Daily 2:00 PM | Anthropic Claude |
| AI candidate analysis | Daily 1:00 PM | Anthropic Claude |
| Sync campaign finance | Weekly (Mon) | OpenFEC |
| Sync elections | Weekly (Mon) | Google Civic |
| Sync voter info | Monthly (1st) | State sources |

## Getting Started

### Prerequisites

- Node.js 18+
- PostgreSQL database (Supabase recommended)

### Installation

```bash
git clone https://github.com/iamscott53/InformedVoter.git
cd InformedVoter
npm install
```

### Environment Variables

Copy the example environment file and fill in your values:

```bash
cp .env.example .env
```

**Required:**

| Variable | Description |
|----------|-------------|
| `DATABASE_URL` | PostgreSQL connection string (with PgBouncer for Supabase) |
| `DIRECT_URL` | Direct PostgreSQL connection (port 5432, for migrations) |
| `ANTHROPIC_API_KEY` | Anthropic API key for AI analysis |

**Optional (for full data sync):**

| Variable | Description |
|----------|-------------|
| `CONGRESS_GOV_API_KEY` | Congress.gov API access |
| `FEC_API_KEY` | OpenFEC campaign finance data |
| `LEGISCAN_API_KEY` | State + federal bill text and roll calls |
| `GOOGLE_CIVIC_API_KEY` | Elections and polling places |
| `UPSTASH_REDIS_URL` | Redis caching URL |
| `UPSTASH_REDIS_TOKEN` | Redis authentication token |
| `CRON_SECRET` | Secret for securing Vercel cron endpoints |

### Database Setup

```bash
npx prisma db push
npm run db:seed
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

### Production Build

```bash
npm run build
npm start
```

## Database Schema

The Prisma schema includes 20+ models organized across these domains:

- **Political:** State, Candidate, CandidatePolicy
- **Legislation:** Bill, BillVote, BillCosponsor
- **Campaign Finance:** CandidateFinance, CandidateTopDonor, CandidateTopIndustry, CandidateExpenditure, IndependentExpenditure, ContributionBySize, ContributionByState
- **Elections:** Election, VoterInfo, VoterInfoDeadline, StatePollingLocator
- **Judicial:** Justice, CourtCase, CaseVote, JusticeFinancialDisclosure, JusticeGift, JusticeReimbursement, JusticeInvestment
- **Users:** User, UserBookmark
